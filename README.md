## 功能特性 / Features

- **完整八字排盘** — 年柱、月柱、日柱、时柱四柱排列，含藏干、十神
- **五行分析** — 金木水火土分布统计与旺衰判断
- **十神解读** — 比肩、劫财、食神、伤官、偏财、正财、七杀、正官、偏印、正印全解析
- **大运排列** — 根据性别与年干阴阳，正确顺排/逆排大运
- **流年运势** — 指定年份的详细运势分析
- **月运分析** — 更精细的月度运势参考
- **合婚分析** — 两人八字配对，看五行互补与地支关系
- **美观命盘** — ASCII art 格式的命盘展示，终端里也很好看

### 学术级准确度

- 以**立春**为年份分界（非农历正月初一）
- 以**节气**为月份分界
- 完整的**五虎遁**（月干推算）和**五鼠遁**（时干推算）
- 准确的**藏干**本气/中气/余气体系
- 地支**三合、六合、六冲、三刑、六害**完整覆盖
- 日柱查万年历，**不编造不确定的信息**

## 八字是什么？/ What is BaZi?

**八字命理**（Four Pillars of Destiny）是中国传统命理学的核心体系，已有千年以上的历史。

"八字"指的是一个人出生时的**年、月、日、时**四个时间点，每个时间点用一个**天干**和一个**地支**来表示，共计八个字，因此称为"八字"。

```
         年柱    月柱    日柱    时柱
天干:     甲      丙      壬      庚
地支:     子      寅      午      戌
         ↑       ↑       ↑       ↑
        出生年   出生月   出生日   出生时
```

通过分析这八个字之间的**五行生克**关系、**十神**互动、**地支**合冲刑害，以及**大运流年**的变化，可以描绘出一个人的性格倾向、人生主题和运势节奏。

这不是迷信，而是一套基于天文历法的**符号分析系统**。就像 MBTI 和星座一样，它提供了一个反思自我的框架。当然，命运始终掌握在自己手中。

## 为什么做这个？/ Why This Exists

2025 年春节期间，中国互联网掀起了一股 "AI 算命" 热潮：

- DeepSeek 被大量算命请求涌入，服务器一度崩溃
- 小红书上 **#deepseek算命** 话题达到 **4200万浏览**
- 各种 AI 命理应用如雨后春笋般出现

然而在 Claude Code 生态中，**命理类 skill 数量为零**。

这个 skill 填补了这个空白。它内置了完整且准确的八字知识体系，让 Claude 能够像一位经验丰富的命理师一样进行分析——同时保持理性、正面、不制造恐惧。

## 设计原则 / Design Philosophy

| 原则 | 说明 |
|------|------|
| **准确** | 天干地支、藏干、五行关系等基础数据经过仔细校对 |
| **诚实** | 不知道就说不知道，不编造日柱，注明不确定性 |
| **正面** | 不说"大凶"、"血光之灾"，提供积极的应对建议 |
| **现代** | 用现代白话文解释，术语首次出现时附带解释 |
| **尊重** | 附带免责声明，不替代专业服务，尊重用户隐私 |

## 贡献 / Contributing

欢迎贡献！以下是一些可以改进的方向：

- **补充特殊格局**：从旺格、从弱格、化气格等
- **完善神煞系统**：天乙贵人、文昌星、桃花星等
- **增加纳音五行**：六十甲子纳音对照
- **改进大运计算**：更精确的节气时间数据
- **多语言支持**：繁体中文、日文、韩文版本
- **添加测试用例**：已知名人八字的排盘验证

### 贡献流程

1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/add-shensha`)
3. 提交更改 (`git commit -m 'Add 神煞 system'`)
4. 推送到分支 (`git push origin feature/add-shensha`)
5. 创建 Pull Request

### 准确性要求

八字命理涉及复杂的传统知识体系，**准确性至关重要**。如果你提交涉及命理知识的 PR，请：

- 注明参考来源（古籍或权威教材）
- 如有流派差异，注明所依据的流派
- 提供可验证的测试案例
